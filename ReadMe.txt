To make the website operational, three steps are required :
- First, you will have to deploy a mongoDB database on this URI : mongodb://localhost:27017
- Second, use a terminal on the backend directory inputing this command : node server.js
- Third, you will have to use a terminal on the forum_jeremy directory, running this command : npm start
    - Then, input y, when it will be indicating that something is already running on the port 3000.
You will be able to access the website on the following adress :
localhost:3001
